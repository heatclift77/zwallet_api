># About
Backend Project Zwallet Apps

># Build With

* Node Js
* Express JS
* **JWT** for Authentication
* **Nodemailer** for Mailer
* **Multer** for Upload file

<br>

># How To Use ?
### Clone This Repo
```
git clone https://github.com/therevolt/Zwallet-Backend.git
```
### Go To Folder
Open GitBash / Terminal
```
cd Zwallet-Backend
```
### Run Project
##### For Development
```
npm run dev
```
##### For Production
```
node app.js
```
